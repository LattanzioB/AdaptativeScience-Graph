# H7.3 — Diagramas

**Épica:** [E7 — Documentación y defensa](README.md)
**Requisito:** Entrega · **Prioridad:** `[SHOULD]` · **Estado:** Pendiente
**Depende de:** E1, E3 · **Habilita:** H7.4

## Historia de usuario
Como evaluador, quiero diagramas de las ontologías, para entender el modelo de un vistazo.

## Criterios de aceptación
- [ ] Diagrama de la **ontología de dominio** (`cipa:`): clases principales y relaciones.
- [ ] Diagrama de la **ontología PBL** (`pbl:`): Points/Badges/Leaderboards, reglas, estado del jugador.
- [ ] (Deseable) Diagrama del **protocolo del diagrama** (H2.1) como DAG.

## Tareas
- [ ] Generar los diagramas (Protégé/OntoGraf, WebVOWL o draw.io).
- [ ] Exportarlos a `docs/diagramas/`.

## Artefactos
- `docs/diagramas/*.png|svg`
