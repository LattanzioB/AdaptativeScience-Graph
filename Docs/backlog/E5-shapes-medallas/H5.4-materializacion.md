## Historia de usuario
Como equipo, queremos escribir `pbl:earnedBadge` a partir del resultado de validación, para poder consultar las medallas obtenidas directamente en E6.

## Criterios de aceptación
- [ ] Un `CONSTRUCT` (o regla) deriva `participant pbl:earnedBadge badge` para cada participante **conforme** a la `BadgeShape` correspondiente.
- [ ] La materialización es **idempotente** y consistente con el resultado de H5.3.
- [ ] Queda documentado el flujo: validar → construir → cargar en GraphDB.

## Tareas

- [ ] Cargar el resultado en GraphDB.

## Notas de modelado
