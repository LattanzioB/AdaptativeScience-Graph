## Historia de usuario
Como equipo, queremos poder derivar, por persona y juego, sus puntos acumulados y su progreso hacia cada medalla, para representar el estado del jugador.

## Contexto (del PDF)
"Estado del Usuario-Jugador: Billetera de Puntos, Colección de Logros, Nivel de Dificultad."

## Criterios de aceptación
- [ ] Para cada `cipa:Persona` se puede obtener la **billetera** (suma de `pbl:PointAward`).
- [ ] Se puede calcular el **progreso** hacia cada medalla (cuánto falta para el umbral).
- [ ] El estado es consultable por **juego particular** (no global), para 6.b.i.
